## ⚙️ Como rodar o projeto

### 1. Instalar as dependências

```bash
npm install
```

### 2. Iniciar o projeto

```bash
npx expo start
```

### 3. Executar o app

Após iniciar, você pode abrir o app em:

* 📱 Expo Go (celular)
* 🤖 Emulador Android
* 🍎 Simulador iOS
